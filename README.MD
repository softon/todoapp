# My Todo App ([Demo](https://softon.github.io/todoapp))

A simple Todo App designed with the help of HTML/CSS & Javascript.

![My Todo App](images/todoapp.png)

## Features
- Add Todo
- Remove Todo
- Mark / Unmark Todo Complete
- Persistent
